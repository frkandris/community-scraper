How it works

1
🔍

Search

For each city and interest topic we run targeted queries on SearXNG, a self-hosted, privacy-respecting meta search engine.

2
🕸

Scrape

Relevant pages are fetched and cleaned using Trafilatura to extract meaningful text — filtering out ads, navigation and noise.

3
🤖

AI Extract

A local Ollama LLM (qwen2.5:7b) reads each page and extracts structured community data: name, schedule, location, contact and more.

4
💾

Store

Results are stored as JSON and versioned with Git, so you can see exactly when each community was discovered and what changed over time.
